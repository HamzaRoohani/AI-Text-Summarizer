Just shipped my first AI project — an AI Text Summarizer!

As someone building my skills in AI/ML development, I challenged myself to build and deploy a complete end-to-end application from scratch.

What it does:
Paste any long article or text → Get a clean, simple summary powered by Google Gemini 2.5 Flash LLM

Tech Stack:
• Google Gemini API (Free LLM)
• Gradio (UI Framework)
• Python + python-dotenv
• Hugging Face Spaces (Free Deployment)

What I learned:
• Working with LLM APIs
• Building UI with Gradio
• Securing API keys with .env
• Deploying apps on Hugging Face Spaces
• End-to-end project structure

Try it live here:
https://huggingface.co/spaces/HamzaRoohani/AI-Text-Summarizer
